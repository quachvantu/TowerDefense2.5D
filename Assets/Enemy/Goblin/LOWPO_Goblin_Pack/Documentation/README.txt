-----------------------------------------------------------
3. TROUBLESHOOTING: "MY MODELS ARE PINK!"
-----------------------------------------------------------
If the models appear bright pink, your project is either
not using the Universal Render Pipeline or the material
shader needs to be refreshed.

A. Ensure URP is Installed:
   Go to Window > Package Manager and make sure
   "Universal RP" is installed.

B. Refresh the Material:
   1. Select the "M_Goblin" material.
   2. In the Inspector, open the Shader dropdown.
   3. Set it to:
      Universal Render Pipeline > Lit (or Simple Lit).

C. Project-Wide Material Conversion (Recommended):
   Go to:
   Edit > Rendering > Materials >
   Convert Selected Built-in Materials to URP.
